## Полная Прозрачность

Zero Token никогда не скрывает переключение. При активации fallback ваш агент работает на бесплатных провайдерах, а не на основной модели. У каждого провайдера своя политика конфиденциальности и хранения данных. **Ознакомьтесь с условиями каждого провайдера перед отправкой конфиденциальных данных.** Скрипт установки чётко показывает, какие провайдеры настроены. Ключи под вашим контролем.

> 💡 Это прозрачный, контролируемый вами failover. Ключи у вас. Провайдеров выбираете вы.
